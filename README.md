## Data Structures
### Arrays 📃
An array is a collection of items stored at contiguous memory locations. They allow for fast access to elements but have a fixed size.

### Linked List 🔗
A linked list is a linear data structure where elements, called nodes, are connected by links. It allows for dynamic size.

### Stack 📦
A stack is a linear data structure that follows the Last In First Out (LIFO) principle. Use it for problems involving recursion!

### Queue ⏰
A queue is a linear data structure that follows the First In First Out (FIFO) principle. Great for scheduling!

### Trees 🌳
Trees are hierarchical data structures. They consist of nodes connected by edges. Binary trees, AVL trees, and more.

### Graphs 🌐
Graphs encompass objects in a network where relationships between objects are represented as edges. Useful in social networks!

## Algorithms
### Sorting Algorithms 🔢
Sorting algorithms arrange data in a particular order. Common algorithms include Bubble Sort, Merge Sort, and Quick Sort.

### Searching Algorithms 🔍
Searching algorithms help find an item in a data structure. Common examples include Linear Search and Binary Search.

### Dynamic Programming 💡
Dynamic Programming is an optimization technique used for solving complex problems by breaking them down into simpler subproblems.
